## 1. はじめに

本システムは、社内ドリンクの在庫管理を行うためのWebアプリケーションです。社員コードでログインし、ドリンクの取り出し・返却を記録できます。

### 動作環境

- モダンブラウザ（Chrome, Firefox, Safari, Edge の最新版）
- PC・タブレット・スマートフォン対応

---

## 2. ログイン

### ログイン方法

1. ブラウザでシステムのURLにアクセスします
2. ログイン画面が表示されます
3. テンキーで **社員コード**（最大8桁）を入力します
4. **ログイン** ボタンを押します

### テンキー操作

| ボタン | 動作 |
|--------|------|
| `0` 〜 `9` | 数字を入力 |
| `←`（バックスペース） | 最後の1文字を削除 |
| `C`（クリア） | 入力をすべて消去 |

### ログインに失敗した場合

- 社員コードが存在しない場合、エラーメッセージが表示されます
- **3回連続で失敗** すると、**5分間ロックアウト** されます
  - ロックアウト中はカウントダウンタイマーが表示されます
  - 時間が経過すると再度ログインできます

### セッションについて

- ログイン後 **30分間** 操作がないと、セッションが切れます
- セッション切れの場合は、再度ログインしてください

## 3. ドリンク取り出し

ドリンクを持ち出す際に記録します。

### 手順

1. ログイン後、ドリンク一覧画面が表示されます
2. 画面上部のモード切替が **「取り出し」**（青色）になっていることを確認します
3. 必要に応じて **お客様名** を入力します（任意）
4. 持ち出したい **ドリンクのタイル** をタップします
5. テンキーで **数量**（1〜9）を入力します
6. 確認画面で内容を確認し、**確定** ボタンを押します
7. 完了画面が表示されたら取り出し完了です

### ドリンクタイルの色の見方

| 色 | 意味 |
|----|------|
| 緑 | 在庫十分（閾値より多い） |
| 黄 | 在庫少なめ（閾値以下） |
| 赤 | 在庫切れ（0本） |

- 在庫が0本のドリンクは取り出しモードでは選択できません
- 在庫数を超える数量は取り出せません

---

## 4. ドリンク返却

取り出したドリンクを返却する場合に使います。

### 手順

1. 画面上部のモード切替を **「返却」**（オレンジ色）に切り替えます
2. 返却する **ドリンクのタイル** をタップします
3. テンキーで **数量** を入力します
4. 確認画面で内容を確認し、**確定** ボタンを押します
5. 完了画面が表示されたら返却完了です

> **補足:** 返却モードでは在庫が0本のドリンクも選択できます。

---

## 5. 完了画面

取り出し・返却が完了すると、完了画面が表示されます。

- 取引内容（ドリンク名、数量、種別、お客様名）が表示されます
- **3秒間のカウントダウン** 後、自動的にドリンク一覧画面に戻ります
- すぐに戻りたい場合は **「戻る」** ボタンを押してください

---

## 6. 統計情報の確認

ドリンクの消費傾向を確認できます。

### アクセス方法

ドリンク一覧画面のヘッダーにある **「統計」** ボタンを押します。

### 表示内容

| セクション | 説明 |
|------------|------|
| 期間選択 | 7日 / 14日 / 30日 / 60日から選択 |
| 在庫枯渇予測 | 現在の在庫・日平均消費量・枯渇までの日数 |
| 日別消費量チャート | 選択期間の日ごとの消費数を棒グラフで表示 |
| ドリンク別内訳 | 円グラフ＋横棒グラフでドリンクごとの消費割合を表示 |
| 上位5ドリンクの日別推移 | 人気ドリンク上位5種の推移を折れ線グラフで表示 |
| 社員別ランキング | 消費数上位20名を横棒グラフで表示 |

## 7. 棚卸し（在庫チェック）

実際の在庫数とシステム上の在庫数を照合する機能です。

### 手順

1. 管理画面の **「棚卸し」** メニューを選択します
2. 各ドリンクについて、実際に数えた **実在庫数** を入力します
   - 「理論在庫」列にシステム上の在庫数が表示されています
   - 「差分」列は自動計算されます
3. すべて入力したら **「確定」** ボタンを押します
4. 結果画面に差分の有無が表示されます

> **注意:** 棚卸しを確定すると、システムの在庫数が実在庫数に更新されます。差分がある場合はWebhookで通知が送信されます。

## 8. よくある質問

### Q. 社員コードを忘れました
管理者に問い合わせてください。

### Q. ドリンクが選択できません
在庫が0本のドリンクは取り出しモードでは選択できません。返却モードであれば選択可能です。在庫の補充は管理者に依頼してください。

### Q. 操作を間違えて取り出し/返却してしまいました
管理者に連絡して、履歴画面で正しい記録になるよう対応を依頼してください。反対の操作（誤って取り出した場合は返却）で在庫を調整することも可能です。

### Q. セッションが切れました
30分間操作がなかった場合にセッションが切れます。再度社員コードでログインしてください。

### Q. ロックアウトされました
3回連続でログインに失敗すると5分間ロックアウトされます。画面に表示されるカウントダウンが終了するまでお待ちください。
